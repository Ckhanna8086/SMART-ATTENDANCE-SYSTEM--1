# 🎓 Smart Attendance System

An AI-powered attendance system that uses **real-time face recognition** via webcam to automatically detect and log attendance — no manual marking needed.

---

## 📌 Features

- 🔍 Real-time face detection and recognition using a webcam
- 📋 Automatic attendance logging to a CSV file (`attendance.csv`)
- ⏱️ Prevents duplicate entries within a 5-minute window per person
- 🟢 Green box for recognized faces, 🔴 Red box for unknown faces
- 💾 Saves face encodings to a `.pickle` file for fast future lookups
